# AI-Powered ATS Resume Analyzer

An AI-powered ATS (Applicant Tracking System) Resume Analyzer built using Python, Streamlit, and Groq LLM APIs. The application analyzes resumes against job descriptions, calculates ATS match scores, ranks multiple resumes, and provides improvement suggestions.

---

## Features

- ATS Resume Match Analysis
- Resume Ranking System
- AI-Based Resume Feedback
- PDF Resume Parsing
- Keyword Extraction
- Multi-Resume Upload Support
- CSV Report Download
- Interactive Streamlit UI

---

## Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3.1
- PDFPlumber
- Pandas
- Regex
- NLP

---

## Run Application

```bash
streamlit run app.py
```

---

## Project Workflow

1. Upload Resume PDF
2. Enter Job Description
3. AI analyzes resume using Llama 3.1
4. ATS score is generated
5. Suggestions and missing keywords are displayed
6. Multiple resumes can be ranked automatically
